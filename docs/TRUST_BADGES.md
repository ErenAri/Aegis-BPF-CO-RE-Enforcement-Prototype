##  Dynamic Badges

For real-time status, you can serve badges from evidence:

```bash
# Create a badge server endpoint
cat evidence/trust_report_*.txt | grep "Score:" | awk '{print $2}'
# Output: 75/80 (93%)

# Use this in your CI to generate dynamic badges
```

---

##  Updating Badges

When you regenerate evidence:

```bash
# Run verification
./scripts/verify_trustworthiness.sh

# Get new score
cat evidence/trust_report_*.txt | grep "Score:"

# Update badges in README with new values
```
